Færdigheder

Jeg er bekendt i de fleste Adobe programmer, både i de enkelte programmer men også i kollaboration med hinanden. Samtidig med kode står jeg også klar med en teoretisk baggrund inde for design teori.

Grafisk Design

Med erfaring indenfor Logo og Plakat design står jeg klar til at producere både store og små projekter, som er tilpasset til dine behov og ønsker.

Prototyper

Ved brug af både teknologiske og fysiske midler kan jeg udfolde flere slags prototyper. Disse prototyper kan ændres og forbedres efter behov.

Konceptudvikling

Med en teoretisk baggrund og viden kan jeg sætte gang i udviklingen af projekter og løsningsforslag.

UI

Med UI kan jeg kreere den visuelle oplevelse som henvender sig til den ønskede kundebase.

UX

For at alt på hjemmesiden kan bevæge sig og skabe handlinger for brugeren, skal der skabes et skelet. Med denne hjemmeside som et eksempel, ses det at jeg kan udføre denne opgave.

HTML5 & CSS

Ved brug af disse kodesprog kan jeg skabe og opbygge hjemmesider som denne. Dette gøres med fokus på hjemmesidens skelet, hvorefter design udseendet også kodes.
